Languru 

Learn to Use Languages via Example Sentences - A Glosbe Client
- Search words, phrases, idioms and find nice use cases of them via real example sentences from life.
- You can learn to use languages like native with this app! 
- It’s also a dictionary, but probably you won’t need it much. Because each example sentence have a translation under it. So you will be able to see which word what means in the context.
- It supports over 100 languages!
- It’s designed to be used fluently in all screen sizes and different font sizes.

Note: This is an unofficial client of glosbe.com and all content is supplied by Glosbe. Definitions, example sentences or highlighted texts in search results may not be fully correct all the time. Because results are gathered from a lot of different sources and some are crowdsourced. So please consider these when using the app.
